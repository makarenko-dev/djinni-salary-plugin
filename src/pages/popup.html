Djinni salary extension

This extension adds a “Get salary” button to every vacancy without a visible salary on djinni.co. When you click it, our server attempts to determine the actual salary set for that vacancy. Your computer doesn’t make any additional requests to Djinni — don’t worry! Once the server finds the salary, it sends the result back and displays it directly on the page.
